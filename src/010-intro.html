Предисловие

Руководство по созданию полноценной электронной музыкальной композиции с использованием дистрибутива Linux под названием KXStudio. Целью этого текста не является ни обзор всех необъятных возможностей дистрибутива, ни замена документации используемых программам. Она сугубо конкретная: показать путь создания музыкального трека от начала до конца, со всеми сложностями и тонкими моментами, как есть - с авторским выбором из множества альтернатив, без учебных упрощений и обобщений.

Руководство находится в процессе разработки. По мере написания/правки текст автоматически публикуется здесь. Исходники на GitHub. Дистрибутив KXStudio можно скачать по этой ссылке.
